Clear float

Algumas vezes você pode querer forçar todos os elementos fiquem alinhados embaixo do outro, para isso você pode usar a propriedade clear.

Sem clear

div1
div2 - note que div2 está depois do div1 no código HTML. Entretanto uma vez que div1 floats está configurado como left, o texto div2 ficará em volta do div1.


Com clear

div3
div4 - Aqui, clear: left; move div4 para baixo do div3, que está com o ,float configurando com left. Isso acontece, porque a propriedade clear: left desabilita o float à esquerda
